# LearnWell Product Portfolio

## Sequence: EEA → Lead Better → MMP-AI → ELT-Dev → FSF

### EEA (Executive Effectiveness Accelerator)
- Audience: CEOs / Founders of 50-249 employee companies
- Format: 1:1 coaching + group sessions
- Duration: 6 months
- Outcome: Executive operating rhythm, delegation framework

### Lead Better
- Audience: Mid-level managers (new to role or struggling)
- Format: Cohort-based, 8 weeks
- Key modules: Feedback loops, 1:1 structure, team health metrics
- NOT called "Manager Momentum" — that name is retired

### MMP-AI (Manager Momentum Platform - AI)
- Audience: Manager cohorts with AI coaching layer
- Includes Dash Agent (AI assistant within the platform)
- Stack: FSF (Vite + React + Supabase)

### ELT-Dev (Executive Leadership Team Development)
- Audience: Intact leadership teams, 250-599 FTEs
- Focus: Alignment, decision-making, strategic execution

### FSF (Full-Stack Framework)
- Internal tool / platform that powers MMP-AI and ELT-Dev
- Stack: Vite + TypeScript + React + shadcn-ui + Tailwind + Supabase
